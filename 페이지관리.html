본 도움말은 디카페 빌더의 특징과 빌더에서 사용되는 몇가지 용어들에 관하여 설명하고 빌더를 이용하여 간단한 사이트를 직접 제작해봄으로써 디카페빌더의 핵심 기능인 페이지 생성과 모듈의 기초적인 사용법을 익히는데 그 목적이 있습니다.

!! 중요 : 도움말의 제작 목적 - 빌더를 이용한 페이지 생성과 모듈의 기초적 사용법에 대한 학습

본 도움말에는 HTML 태그 사용법이나 웹사이트 제작에 관한 방법론 등은 포함되어 있지 않습니다.

따라서 작업자는 이미 HTML 태그 사용법이나 웹사이트 제작 방법 등에 관하여 충분히 숙지하고 있음을 전제로 하고 본 도움말이 작성되었음을 알려드립니다.

디카페 빌더의 특징

1-1. 페이지 단위의 사이트 제작
정하여진 Site Map을 기준으로 페이지(디렉토리)를 먼저 생성한 후 페이지 단위로 사이트를 제작하실 수 있습니다.

1-2. 제작한 페이지의 연결
페이지에는 작업자가 지정한 코드를 페이지 이름으로 하는 경로가 자동 부여됩니다. 경로에는 템플릿 변수가 포함되어 있습니다. (예:{$변수}/index.php) 생성된 페이지들은 Anchor 태그를 사용하여 연결합니다. (예: 메뉴)

1-3. 모듈제공
디카페가 제공하는 다양한 모듈을 활용하면 별도의 프로그래밍 작업 없이도 게시판, 회원관리, 일정관리 등을 손쉽게 구현할 수 있으며, 또한 작업자 스스로 필요한 모듈을 제작하여 활용하실 수도 있습니다.

1-4. 방문자수 통계
홈페이지에 방문자수를 표시하시려면 아래의 변수를 에디터에 삽입하세요
 변수: {{$TODAY_COUNT}}
오늘날짜에 방문자 방문자 표시 합니다.
 변수: {{$TOTAL_COUNT}}
전체 방문자 표시 합니다.


1-4. 회원수 통계
홈페이지에 회원수를 표시하시려면 아래의 변수를 에디터에 삽입하세요
 변수: {{$TODAY_MEMBER_COUNT}}
오늘날짜에 가입한 회원수 표시 합니다.
 변수: {{$TOTAL_MEMBER_COUNT}}
가입된 전체 회원수 표시 합니다.
